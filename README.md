# Mesh Samples

This directory holds Bluetooth Mesh sample applications built on the nRF Connect SDK (Zephyr
Bluetooth Mesh). Each subfolder is a self-contained project with its own build instructions and
documentation.

> **IMPORTANT:** These samples are not intended to be maintained actively!

## Samples

- **provisioner_network_setup** - Scenario-based Mesh provisioner: PB-ADV provisioning plus
  deployment rules (model binding, publication, subscription); see
  [provisioner_network_setup/README.md](provisioner_network_setup/README.md).
